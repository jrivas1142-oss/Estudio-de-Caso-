Este proyecto consistió en el análisis y rediseño de un sitio web existente (Burger King) para solucionar sus deficiencias de adaptabilidad a diferentes dispositivos. La actividad se basó en la metodología Mobile-First, priorizando la experiencia en pantallas móviles.

Se identificaron problemas críticos en la navegación (desbordamiento de enlaces) y en el layout de tarjetas de menú (falta de escalado). Para solucionarlos, se desarrolló una nueva estructura utilizando HTML5 semántico.

En el CSS, se implementaron:

    Flexbox para alinear elementos del header y manejar la estructura del menú hamburguesa en móvil.

    CSS Grid para el layout principal de las tarjetas de menú, permitiendo que se apilen en una sola columna en móvil y escalen a 3 o 4 columnas en tablet y escritorio.

    Media Queries con puntos de ruptura definidos para tablet (769px) y escritorio (1024px), ajustando el diseño en cada vista.

El resultado es un sitio web fluido y funcional que cumple con los estándares modernos de desarrollo, mejorando significativamente la usabilidad y la visualización en las tres vistas comprobadas.
